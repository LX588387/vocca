# 示例

每个脚本都可直接运行（`python examples/<name>.py`），只依赖已安装的 vocca 核心。

| 脚本 | 演示内容 |
|------|----------|
| `quickstart.py` | 最简合成：一句话 → WAV |
| `streaming_demo.py` | 流式生成，并验证与整段离线结果逐样本一致 |
| `control_matrix.py` | 同一句话在不同情感 / 风格下的输出差异 |
| `prompt_tags.py` | 用 `[key:value]` 内联标签指定控制条件 |

> 提示：参考实现默认随机初始化权重，生成的是可复现的「合成音」而非自然语音。
> 示例的重点是展示 **API 与机制**（控制、流式、提示标签），而非音质。
