# 安全策略

## 支持的版本

| 版本 | 是否维护 |
|------|----------|
| 0.3.x | ✅ |
| < 0.3 | ❌ |

## 报告漏洞

如果你发现了安全问题，请**不要**直接提交公开 issue。请通过 GitHub 的私密
安全公告（Security Advisory）功能，或私信仓库维护者报告。

请在报告中尽量包含：

- 问题的类型与影响范围
- 复现步骤或 PoC
- 相关的版本 / 环境信息

我们会在合理时间内确认收到，并在修复后于 `CHANGELOG.md` 中致谢（如你愿意）。

## 说明

vocca 是一个纯计算库，不涉及网络服务；主要的安全关注点在于加载不受信任的
权重 / 配置文件。请只从可信来源加载 `.npz` 权重与 YAML 配置。
